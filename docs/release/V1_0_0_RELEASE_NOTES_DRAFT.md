# Odin Agent Shell v1.0.0 Release Notes Draft

Odin Agent Shell v1.0.0 is prepared as a local-first, candidate-only candidate-work kernel for bounded AI coordination, receipt discipline, and app-owned apply workflows.

## Highlights

- Local-first Universal Work and Candidate Artifact workflow.
- Candidate-only outputs with app-owned apply, persistence, state mutation, and external sends.
- Deterministic validators and evidence reports for release-readiness lineage.
- Operational spine and semantic kernel surfaces for bounded coordination.
- Local provider receipts remain disabled by default unless a host explicitly enables them.
- Critic, runtime, and evidence boundaries preserve the rule that model output is projection, not truth.

## Manual Release Boundary

This draft does not claim an external release. No external release claim exists until a maintainer actually creates and verifies any tag, GitHub Release, PyPI publication, and release assets.

## Non-Claims

- No production readiness claim.
- No security certification claim.
- No model benchmark claim.
- No model superiority claim.
- No external release claim until maintainer action creates and verifies external release facts.
