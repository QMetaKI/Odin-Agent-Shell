## Maria/Michael Superposition

The QLI Master Interface selects a Maria/Michael profile:

- 80/20 default for general holding/coherence/builder work
- 20/80 for compiler, code, validator, claim-boundary-heavy work
- 50/50 for public/reality-balanced output
- 35/65 for incident, risk, conflict, debug and fallback
- 85/15 for story/listening/intake flows

Maria/Michael is not persona. It is scoring posture.
