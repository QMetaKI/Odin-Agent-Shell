# Shadow Runtime State Machines v7.1

## Universal Work Shadow State Machine

```text
SHADOW_WORK_RECEIVED
→ SHADOW_BINDING_CHECKED
→ SHADOW_OUTPUT_CONTRACT_CHECKED
→ SHADOW_ARTIFACT_LENS_SELECTED
→ SHADOW_CONTEXT_CAPSULE_CREATED
→ SHADOW_BUS_BATCH_STARTED
→ SHADOW_WORKLET_GRAPH_CREATED
→ SHADOW_SLOT_CONTRACT_CREATED
→ SHADOW_MODEL_ROUTE_PLANNED
→ SHADOW_CANDIDATE_CREATED
→ SHADOW_DNA_CREATED
→ SHADOW_RESPONSE_PACKET_CREATED
→ SHADOW_FINAL_GATE_PASSED
```

Failure transitions:

```text
SHADOW_BINDING_REJECTED
SHADOW_OUTPUT_CONTRACT_REJECTED
SHADOW_FORBIDDEN_ACTION_REJECTED
SHADOW_PRIVACY_REJECTED
SHADOW_ROUTE_REJECTED
SHADOW_FINAL_GATE_BLOCKED
```

## Semantic Bus Shadow State Machine

```text
BUS_DISABLED
→ BUS_LOCAL_START_REQUESTED
→ BUS_LOCAL_ACTIVE
→ BATCH_OPENED
→ EVENT_ACCEPTED
→ EVENT_RECORDED
→ BATCH_CLOSED
→ REPLAY_AVAILABLE
```

Failure transitions:

```text
BUS_NETWORK_REQUEST_BLOCKED
BUS_INVALID_CHANNEL_BLOCKED
BUS_CONTEXT_PRIVACY_BLOCKED
BUS_APP_MUTATION_BLOCKED
```

## Candidate Shadow State Machine

```text
CANDIDATE_INIT
→ CONTENT_PROJECTED
→ CLAIM_STATUS_ASSIGNED
→ DNA_LINKED
→ ACTIONS_MARKED_APP_OWNED
→ FINAL_GATE_READY
→ RESPONSE_PACKET_READY
```

Failure transitions:

```text
CANDIDATE_DIRECT_APPLY_BLOCKED
CANDIDATE_UNSUPPORTED_CLAIM_BLOCKED
CANDIDATE_MISSING_TRACE_BLOCKED
```

## Model Route Shadow State Machine

```text
ROUTE_REQUESTED
→ RESOURCE_PROFILE_READ
→ LATENCY_MODE_READ
→ PRIVACY_POLICY_READ
→ DEFAULT_ROUTE_SELECTED
→ ESCALATION_CHECKED
→ ROUTE_PLAN_CREATED
```

Failure transitions:

```text
REMOTE_ROUTE_BLOCKED
HEAVY_ROUTE_BLOCKED_BY_LATENCY
ROUTE_EXCEEDS_POLICY
LOW_MEMORY_DEGRADE_REQUIRED
```
